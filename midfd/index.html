FDライクな操作感を目指した、Windows向けの軽量ファイラー

キーボード主体で、ファイル操作を素早く進める。
Explorerとコマンドラインの間で使える軽量なGUI補助ツールです。

通常版をダウンロード 配布元ページを見る
動作要件: Windows x64 / .NET 10 Desktop Runtime x64 が必要です

Runtime が不足している場合は 導入ガイド を確認してください。

本体画面 ファイル一覧の見え方を確認できます。

ダウンロード / 動作要件

通常版

MidFD-win-x64.zip

ZIP を展開して MidFD.exe を起動するだけで使い始められます。インストール不要です。

通常版をダウンロード 配布元ページ (GitHub Releases)

動作要件

  • OS: Windows x64
  • Runtime: .NET 10 Desktop Runtime x64

起動時に「Download it now」ボタンが表示された場合は、そこからRuntime を追加できます。それ以外の方法は 導入ガイド を参照してください。

MidFDの特徴

キーボード主体の軽快操作

移動、選択、ファイル操作、コマンド実行など、ファイラーの全主要操作をキーボードだけで軽快に行えます。設定からコマンド単位のキーカスタマイズやFunctionキー割り当ても可能です。

移動と管理の高速化

複数タブとカテゴリ機能で作業フォルダを分離。よく使うフォルダへ素早く移動できるQuickAccessや、マーク状態を保存・復元するMarkSlotなどの強力な整理機能を搭載しています。

メディアプレビューと外部連携

動作の軽い内蔵画像ビューアのほか、ffmpegと連携した動画静止画プレビュー（VideoStill）を内蔵。7-Zipなどの使い慣れた外部ツールを直接呼び出せるAlt slot launcherもサポートします。

クイックスタート

まずは以下の主要な基本キーを覚えるだけで、MidFD の操作を始められます。

キー動作
↑ / ↓選択項目の上下移動
Enterファイルを開く、または内蔵ビューアで表示
Backspace親フォルダへ移動
L移動履歴（Logdsk）からディレクトリを素早く移動
Qよく使う場所（QuickAccess）を開く
Ctrl+Shift+Pコマンドパレットを開き、機能や外部ツールを検索して実行

外部ツールとの柔軟な連携

外部ツールは初回起動の必須条件ではありません。 必要になったら後から設定の「外部連携」タブに登録し、MidFDから素早く呼び出すことができます。

Archive
