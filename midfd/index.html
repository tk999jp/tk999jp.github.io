MidFD

Windows向けの軽量・高速なタブ型ファイラー

シンプルで軽快な操作感を追求した、C# & .NET 10 製ファイルマネージャー。必要な機能に素早くアクセスできるキー操作やコマンドパレットを搭載しています。

通常版ダウンロード (win-x64) GitHub
※動作要件: .NET 10 Desktop Runtime

主な特徴

軽快な動作性能

.NET 10 テクノロジーを基盤とし、起動速度と操作レスポンスの軽快さを追求しました。

タブ型インターフェース

複数のフォルダを並行して切り替えながら、スムーズに整理・移動が可能です。

無駄のないシンプル設計

複雑な多機能化を避け、直感的かつ確実にファイル操作を行える画面構成です。

ダウンロードに関する注意点

GitHub リポジトリ上にある 「Source code (zip)」 は、開発者向けのソースコード一式です。ビルドしないと実行ファイルが含まれていないため、通常利用目的の方は上記の「通常版 MidFD (win-x64.zip)」または GitHub Releases のビルド済みバイナリをご利用ください。

動作環境と前提要件

動作には .NET 10 Desktop Runtime が必要です

MidFD.exe 起動時に .NET Runtime 不足のダイアログが表示された場合は、画面の「Download it now」ボタンを押すことで簡単にインストーラーをダウンロードできます。

起動時の流れや、うまくインストールできない場合の対処方法はガイドをご確認ください。

サポート・FAQへ

Runtimeの導入でお困りの場合

PCの環境制限等により .NET 10 Desktop Runtime をシステムに直接インストールできない場合は、ランタイムがあらかじめ同梱された「自己完結型 (Self-Contained)」の配布パッケージの利用をご検討ください。

※自己完結型パッケージは、ファイルサイズが大きくなります。可能な限り通常版＋システムへのRuntime導入を推奨します。

→ GitHub Releases から Self-Contained 版 (MidFD-win-x64-selfcontained.zip 等) を探す
