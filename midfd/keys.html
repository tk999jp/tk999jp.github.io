Keybindings Reference

キー操作ガイド

MidFD はキーボードでの素早い操作を重視して設計されています。
「標準」と「FD互換」の2つのキー操作体系をサポートしており、設定から好みの割り当てに変更できます。

最初に覚える基本キー

まずは以下の基本操作を覚えると、すぐに MidFD を使い始めることができます。

機能 キー 説明
開く / 表示 Enter 選択項目を開く、または内蔵ビューアで画像・動画を表示します。
関連付けで開く Z OS の関連付け（既定のアプリ）でファイルを開きます。
外部エディタで開く E / F4 選択ファイルを登録した外部エディタで開きます。
コマンド実行 X コマンド実行ダイアログを開きます。
コマンドパレット Ctrl + Shift + P 機能や外部ツールをすばやく検索・実行します。
移動履歴 (Logdsk) L / F9 過去の移動履歴からディレクトリを素早く移動します。
お気に入り (QuickAccess) Q / F8 登録済みの場所や最近使ったフォルダへ移動します。

ファイル一覧（Browser）操作

カーソル・移動

操作 キー 説明
上下移動 ↑ / ↓ 選択項目を上下に移動します。
左右移動（列） ← / → 複数列表示時、選択項目を左右の列へ移動します。
ページ移動 PageUp / PageDown 一覧をページ単位でスクロールします。
親ディレクトリへ移動 Backspace / Alt + ↑ 現在位置の1つ上のフォルダへ移動します。
ディレクトリ履歴を戻る Alt + ← / Ctrl + Backspace 移動履歴を1つ戻ります。
ディレクトリ履歴を進む Alt + → 移動履歴を1つ進みます。
ドライブのルートへ \（バックスラッシュ/円マーク） 現在のドライブの最上層（ルート）へ移動します。
パスの直接入力 Ctrl + L パス入力欄を開いて直接フォルダパスを指定します。

ファイル編集・操作

操作 キー 説明
名前変更 (Rename) R / F2 選択しているファイルやフォルダの名前を変更します。
コピー (Copy) C / F3 コピー先の入力欄を開いてコピーを実行します。
移動 (Move) M / F5 移動先の入力欄を開いて移動を実行します。
削除 (Delete) D / Delete 選択またはマークしている項目を削除します。
新規フォルダ作成 K 新規フォルダ作成ダイアログを開きます。
新規ファイル作成 N 空の新規ファイル作成ダイアログを開きます。
属性・日時変更 A ファイルの属性（読み取り専用・隠し等）や日時を一括変更します。
圧縮 P 選択・マークした項目を ZIP などに圧縮します。
解凍 U 選択した圧縮ファイルを展開・解凍します。
元に戻す / やり直し Ctrl + Z / Ctrl + Y 直前のファイル操作を元に戻す、またはやり直します（対応操作のみ）。

表示・整理

操作 キー 説明
再読込 (Reload) Ctrl + R / Shift + R 現在フォルダの内容を最新に更新します。
フィルタ F / Ctrl + F ファイル名で一覧を絞り込みます。
ソート順変更 F6 ソート順変更ダイアログを開きます。
ディレクトリツリー T 視覚的なディレクトリツリーを開きます。
表示モード切替 Ctrl + 1 / 2 / 3 「ファイル名のみ」「ファイル名＋サイズ」「詳細情報」を切り替えます。
システム情報 I ドライブ使用量やメモリなどのシステム状況を表示します。
設定を開く O / Alt + F5 設定ダイアログを開きます。

マーク操作と MarkSlot

MidFD では、複数のファイルを選択（マーク）して一括でコピー、移動、削除などの操作を行えます。

操作 キー 説明
マーク切り替え Space / Insert カーソル位置の項目のマーク状態を反転させて下に移動します。
ファイルのみ全選択 Home ディレクトリ（フォルダ）を除外し、ファイルのみを全選択/解除します。
フォルダを含む全選択 End / Ctrl + A すべてのフォルダとファイルを一括選択/解除します。
マークスロット画面を開く Ctrl + M / F11 現在のマーク状態をスロットに一時保存・復元する画面を開きます。

ファンクションキー割り当て

操作設定のプリセットにより、ファンクションキー（F1〜F12）の初期機能が変化します。キーおよびファンクションバーの割り当ては設定からカスタマイズ可能です。

物理キー 標準プリセット FD互換プリセット
F1Help（ヘルプ表示）Help（ヘルプ表示）
F2Rename（名前変更）Check / Execute（実行・決定）
F3Copy（コピー）Copy（コピー）
F4外部エディタで開くEdit（外部エディタで開く）
F5Reload（再読込）Rename（名前変更）
F6Sort（並び替え）Sort（並び替え）
F7Filter（フィルタ）Filter（フィルタ）
F8QuickAccess（お気に入り）Tree（ツリー表示）
F9Logdsk（ディレクトリ移動）Logdsk（ディレクトリ移動）
F10Command PaletteUnpack（圧縮ファイル解凍）
F11MarkSlot（マークスロット）Top（一覧の最上部へ移動）
F12コマンド一覧の表示Bottom（一覧の最下部へ移動）

メディアプレビューと外部連携

動画静画プレビュー (VideoStill) 操作

内蔵画像ビューアで動画プレビュー（ffmpeg 連携時）を表示している際、以下のシーク操作を行えます。

操作 キー 説明
シーク（戻る/進む） ← / → 動画の再生位置をステップ移動します（動画長に応じて自動調整）。
大シーク（戻る/進む） Shift + ← / Shift + → 大きくステップ移動します。
先頭に戻る Home 動画の最初（0秒）に位置を戻します。
位置追従で外部再生 Ctrl + Enter 現在のプレビュー位置を引き継いで外部プレーヤー（ffplay等）で再生します。

外部ツール Alt slot launcher

設定で外部ツールにショートカットキー（Alt + 英数字）を割り当てると、ファイラーから一発で起動できます。

操作 キー 説明
ランチャー表示 Alt（単押し） 登録済みの外部ツール一覧スロットを画面に表示します。
スロット直接起動 Alt + 英数字 対応するスロットに登録した外部ツールを直接起動します。
選択して起動 ↑ / ↓ を経て Enter Alt で開いたランチャー内で選択を移動し、決定して起動します。

より詳細な割り当てや拡張機能のキー操作については、GitHub のリファレンスドキュメントをご覧ください。
