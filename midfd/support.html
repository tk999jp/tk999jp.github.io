注意事項・FAQ・サポート

MidFD を安全かつ快適にご利用いただくための注意事項とサポートに関する情報です。

重要：ご利用にあたっての注意事項

データのバックアップについて

MidFD はファイル管理・操作ソフト（ファイラー）です。ファイルの移動、コピー、削除などの処理を行います。
万が一の不具合によるデータ消失を防ぐため、重要なファイルは必ず事前、または定期的にバックアップを取ってからご利用ください。本ソフトを使用した結果生じたデータ損害等に関して、開発者は一切の責任を負いかねます。

サポートポリシーについて

本ソフトウェアは個人開発のフリーウェアです。無償でご利用いただけますが、個別サポート（メール、返信保証のある対応など）の保証は行っておりません。予めご了承ください。

よくあるご質問 (FAQ)

Q. 7-Zip, ffmpeg などのツールは同梱されていますか？
A.
同梱していません。
書庫ファイルの圧縮・展開機能や、メディア情報の取得機能などで使用する外部ツール（7-Zip, ffmpeg, ffprobe, ffplay など）は本アプリには同梱していません。
これらの機能を利用される場合は、各公式サイト等から別途ツールを入手し、PCの環境変数 PATH に通すか、アプリ設定等で実行ファイルへのパスを登録してください。
Q. アプリが起動しない、エラーダイアログが出る
A.
.NET 10 Desktop Runtime が必要です。
お使いのPCに動作要件を満たすランタイムがインストールされているか確認してください。詳細は .NET 10 Desktop Runtime 導入ガイド をご確認ください。

不具合報告・要望について

MidFD の不具合（バグ）の報告や機能要望は、GitHub の Issue にて受け付けています。
報告される際は、発生した現象の具体的な手順やエラーメッセージ、動作環境を明記していただけますと助かります。

  • 不具合の報告・要望の送付先：
    GitHub Issues (tk999jp/MidFD)
  • 詳細なサポートドキュメント（日本語）：
    UserDocs/SUPPORT.md (GitHub)
